# animolto
Another modding tool (with CUI and GLI support) I made for my Italian friend. 

Automatically **scales animations from Budokai Tenkaichi 2 & 3** (PS2 version only) in the same fashion as Ginyu's animations after using Body Change.

Gone are the days of going in Blender and **manually resizing** one character model to match the other in order to get the **required scaling coefficient/multiplier**.

## Usage/Demonstration - CLI
The first argument is the **costume PAK** of the character **subject to the animation scaling**.

The second argument is the **costume PAK** of the character **whose animations will be scaled**.

The third argument is a directory containing the **animations that will be scaled**.

Simply put, the animations of one character (the source) will be **adjusted to match the coordinates of another character** (the target).

In the example below, Dabura's animations are resized to match Golden Great Ape Baby Vegeta's proportions.
